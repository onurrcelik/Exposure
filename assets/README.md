# Assets Directory

This directory contains all source assets for the Exposure project.

## Structure

```
assets/
├── brand/              # Brand assets and color palettes
├── favicon/            # Favicon source files
└── logos-source/       # Logo source files
    ├── logos/          # Main logos (Exposure brand)
    └── carousel/       # Partner/company logos for carousel
```

## Important Notes

- **DO NOT** use files from this directory directly in the app
- All public-facing assets should be in the `/public` folder
- This folder is for source files and organization only
- Files here should be copied to `/public` when needed for the app
